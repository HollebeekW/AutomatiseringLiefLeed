voor email testing is gebruik gemaakt van mailtrap.io

Om dit zelf te testen:
1. Ga naar mailtrap.io en maak een account aan.
2. Ga naar Sandbox -> Inboxes en maak evt. een nieuwe inbox aan.
3. Ga naar Integration -> SMTP.
4. Vul uw eigen username en password in, in appsettings.json onder "SmtpSettings"
